# Average True Range (ATR) Explained

The Average True Range (ATR) is a technical indicator introduced by J. Welles Wilder Jr. that measures market volatility by decomposing the entire range of an asset price for that period.

## Calculating the True Range

The foundation of ATR is the True Range (TR), which accounts for overnight gaps and limit moves that a simple high-low range might miss. TR is defined as the greatest of the following:

1. Current High minus Current Low
2. Absolute value of Current High minus Previous Close
3. Absolute value of Current Low minus Previous Close

<ConceptCard title="ATR Calculation">
  The ATR is an exponential moving average (or smoothed moving average) of the True Range over a specified number of periods (commonly 14).
  
  `ATR_t = ((ATR_{t-1} * 13) + TR_t) / 14`
</ConceptCard>

## Interpreting ATR Values

ATR is absolute, not relative. A higher ATR indicates higher volatility, while a lower ATR indicates lower volatility. However, ATR is proportional to the asset's price; a $2000 asset will inherently have a larger ATR than a $50 asset, assuming similar percentage volatility.
